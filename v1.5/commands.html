Command Line Reference

명령어 사용가능한 옵션
bundle check
Gemfile의 리스트가 현재 인스톨 된 젬의 의존성을 만족하는지 체크
--gemfile
--path
bundle clean
번들러 디랙토리에서 사용하지 않는 젬들을 청소
--force
bundle config
설정 값을 보거나 지정
bundle console
번들이 프리로드된 IRB세션 열기
bundle exec
번들의 컨택스트에서 명령어 실행
bundle gem
루비젬을 만들기위한 탬플릿 생성
--bin or -b
bundle help
각 서브 명령어의 자세한 도움말 표시
bundle init
현재 디랙토리에 Gemfile 생성
--gemspec
bundle install
시스템에 현재 환경을 인스톨
--binstubs
--clean
--deployment
--frozen
--full-index
--gemfile
--local
--no-cache
--no-prune
--path
--quiet
--shebang
--standalone
--system
--without
bundle open
주어진 번들 젬의 소스 디랙토리 열기
bundle outdated
새로운 버전이 나온 설치된 젬의 리스트
--local
--pre
--source
bundle package
vendor/cache에 모든 젬을 케쉬, 락
--no-prune
--all
bundle platform
플렛폼 호환성 정보 표시
--ruby
bundle show
번들에 포함된 모든 젬의 리스트나, 특정 젬의 경로를 표시
--paths
bundle update
현재 환경을 업데이트
--local
--source
bundle version
버전 정보 출력
bundle viz
비쥬얼 의존성 그래프 생성
--file or -f
--format or -F
--requirements or -r
--version or -v
